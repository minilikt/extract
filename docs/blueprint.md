# **App Name**: MediaSifter

## Core Features:

- CSV File Import: Allow users to upload a CSV file containing image and GIF URLs.
- Automatic Media Extraction: Automatically scan the CSV and use a tool to extract valid image and GIF URLs.
- Media Preview: Display the extracted images and GIFs in a preview gallery.
- Bulk Download: Enable users to select and download multiple images/GIFs at once.
- Customizable File Renaming: Allow users to customize the file names using data from the CSV columns before downloading.
- Media Type Filtering: Provide options to filter media by type (image or GIF).

## Style Guidelines:

- Primary color: Saturated electric purple (#BE63FF) to convey technical capability.
- Background color: Dark charcoal (#222222) to create a sleek dark theme.
- Accent color: Vibrant cyan (#63FFDA) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable feel.
- Use clean, minimalist icons for file types and actions.
- Implement a grid-based layout for media previews.
- Use subtle animations to indicate loading and selection states.
